TechShop Order Processor

Backend Corporativo com Validação de Regras de Negócio

O Desafio

Simular o backend de um e-commerce de grande porte, onde é necessário processar milhares de pedidos, validar estoque, calcular fretes complexos e gerar relatórios gerenciais, garantindo a integridade dos dados.

A Solução

Desenvolvi um processador de pedidos robusto em .NET Core. O foco foi o uso de LINQ para manipular coleções de dados de forma performática e a implementação de classes de serviço dedicadas para cada regra de negócio.

Destaques Técnicos:

  • LINQ Avançado: Utilizado para filtrar pedidos, agrupar vendas por categoria e calcular totais.
  • Validação: Sistema que impede vendas de itens sem estoque ou com dados inválidos.
  • Relatórios: Geração automática de arquivos CSV com o resumo diário das operações.

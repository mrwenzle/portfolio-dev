English Dictionary App

Aplicação Desktop WPF com Arquitetura MVVM

O Desafio

Criar uma ferramenta de estudo de inglês rápida e offline-first, que não dependesse de navegadores pesados. O objetivo técnico era implementar uma separação clara entre a interface (View) e a lógica (ViewModel).

A Solução

Desenvolvi uma aplicação WPF (.NET 8) utilizando o padrão MVVM com a biblioteca CommunityToolkit.Mvvm. Isso permitiu criar uma interface reativa, onde a tela atualiza automaticamente quando os dados mudam, sem precisar de código "spaghetti" no backend da View.

Funcionalidades Principais:

  • Busca de definições em tempo real via API REST.
  • Interface moderna utilizando XAML e Material Design.
  • Tratamento de erros e estados de carregamento (Loading Spinner).
  • Pronúncia de áudio integrada.

Tech Stack

C# .NET 8 WPF MVVM Toolkit Rest API XAML
